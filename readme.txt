Optimize your blog SEO by converting tags to meta keywords that will display in your website code

== Description ==
Does your WordPress blog rank poorly in SEO? Is it missing Keywords in the header? Forget fancy SEO plugins, use this simple plugin to convert tags for individual posts and pages to meta keywords that will be picked up by search engines

== Installation ==
You can directly install the plugin from the WordPress repository or you can upload the zipped plugin through the Plugin section of the admin dashboard. Activate the plugin and you can see the result in the output HTML
